Tic Tac Toe game using Python

The Tic Tac Toe game is a classic two-player game where players take turns marking X's and O's in a 3x3 grid. The player who succeeds in placing three of their marks in a horizontal, vertical, or diagonal row wins the game. In this project, we will create a simple Tic Tac Toe game using Python programming language. To begin with, we will create a 3x3 grid using a nested list in Python. We will represent the X and O marks using the characters 'X' and 'O', respectively. We will then display the grid on the console and prompt the first player to make their move.
